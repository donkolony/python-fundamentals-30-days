# List Methods

## ðŸŸ¢ Methods that **modify the list**

- `append(item)` â†’ Add item at the end
- `insert(index, item)` â†’ Add item at a specific index
- `remove(item)` â†’ Remove the first occurrence of item
- `pop(index)` â†’ Remove item at specified index (or last item if index not provided)
- `del list_name[index]` â†’ Delete item at index (or entire list if no index)
- `clear()` â†’ Empty the list
- `extend(iterable)` â†’ Append all elements from another list/iterable
- `reverse()` â†’ Reverse the list in place
- `sort(reverse=False)` â†’ Sort list in place (default ascending, `reverse=True` for descending)

## ðŸ”µ Methods that **do not modify the list**

- `copy()` â†’ Return a shallow copy of the list
- `count(item)` â†’ Return number of times an item appears
- `index(item)` â†’ Return index of the first occurrence of item
- `sorted(list, reverse=False)` â†’ Return a **new sorted list** (original unchanged)
- `+` (plus operator) â†’ Concatenate two lists and return a new one
